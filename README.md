## Run Locally

Clone the project

```bash
  git clone https://github.com/AgusT613/ecommerce-react
```

Go to the project directory

```bash
  cd ecommerce-react
```

Install dependencies

```bash
  npm install
```

If you use nvm (Node Version Manager)

```bash
  nvm install <node-version>
  nvm use <node-version>

  npm install
```

Start the server

```bash
  npm run dev
```
